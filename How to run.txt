How to run:
- be sure you have xampp installed
- ensure that there's a xampp folder in C:
- all files are in Xampp > htdocs > foldername
- start apache and mysql in xampp
- type localhost/foldername/login.php sa search bar
- type localhost/phpmyadmin then import yung sql file
- pag nagsign up lalabas sa "users" ng log_in_db yung bagong entry
- select entry if gustong idelete
